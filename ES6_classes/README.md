# ES6 Classes

This project covers ES6 classes, inheritance, getters, setters, static methods, and metaprogramming.

## Learning Objectives

- How to define a Class
- How to add methods to a class
- Why and how to add a static method to a class
- How to extend a class from another
- Metaprogramming and symbols

## Requirements

- All files interpreted/compiled on Ubuntu 20.04 LTS using node 20.x.x and npm 9.x.x
- All files should end with a new line
- Code should use the `.js` extension
- Code will be tested using Jest
- Code will be verified against lint using ESLint

## Setup

```bash
npm install
```

0. **0-classroom.js** - ClassRoom class

7. **7-airport.js** - Airport class
